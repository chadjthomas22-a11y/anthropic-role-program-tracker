# Anthropic Role Tracker

Interactive dashboard for tracking responsibilities, KPIs, and progress against the **Head of Programmatic Outcomes — Partners** role at Anthropic.

## Features

- **8 responsibilities** mapped directly from the job description
- **32 KPIs** with editable progress tracking and visual progress bars
- **Movable target dates** for each responsibility
- **Status tracking**: Not started, In progress, On track, At risk, Blocked, Complete
- **Priority tags**: Critical path, High, Medium
- **"Where to start" view** with recommended 90-day sequencing
- **KPI tracker view** for dense progress monitoring
- **Local storage persistence** — your data survives browser refreshes

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
npm run build
git add -A
git commit -m "deploy"
git push origin main
```

Then in your GitHub repo settings, set Pages source to **Deploy from a branch** → `main` → `/docs`.

Your site will be live at `https://YOUR_USERNAME.github.io/anthropic-role-tracker/`
